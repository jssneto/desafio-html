O HTML (Hypertext Markup Language) é a linguagem de marcação usada para criar páginas da Web. Com HTML, você pode criar documentos que podem ser exibidos em um navegador da Web e acessados em qualquer lugar do mundo.

Tags HTML

Conforme o curso, aprendemos várias Tags que levaremos para vida, que nos ajudarão a construir nossas carreras no mundo da programação Web. Abaixo seguem as tags que aprendemos no curso da DIO.ME

  • <h1> a <h6>: para cabeçalhos de diferentes níveis;
  • <p>: para parágrafos de texto;
  • <mark>: para destacar texto;
  • <small>: para texto em tamanho reduzido;
  • <i>: para texto em itálico;
  • <u>: para texto sublinhado;
  • <ol>: para listas ordenadas;
  • <ul>: para listas não ordenadas;
  • <li>: para itens de lista;
  • <a>: para links;
  • <hr>: para uma linha horizontal;
  • <sub>: para texto sobrescrito;
  • <sup>: para texto subscrito;
  • <blockquote>: para citações em bloco;
  • <font>: para ajustar a fonte do texto;
  • <del>: para texto tachado;
  • <abbr>: para abreviações.

FORMULÁRIOS: O que são Formulários?

Formulários HTML são um dos principais pontos de interação entre um usuário e um web site ou aplicativo. Eles permitem que os usuários enviem dados para o web site. Na maior parte do tempo, os dados são enviados para o servidor da web, mas a página da web também pode interceptar para usá-los por conta própria.

Um formulário HTML é feito de um ou mais widgets. Esses widgets podem ser campos de texto, (linha única ou de várias linhas), caixas de seleção, botões, checkboxes ou radio buttons. A maior parte do tempo, estes elementos são emparelhados com uma legenda que descreve o seu objetivo.
